# 任务调整模板

你可以直接按下面格式发我：

```text
任务调整：
- 任务标题：
- 所属项目：
- 父任务：（标题不唯一时填写）
- 调整类型：状态 / 时间 / 项目 / 标题 / 备注 / 优先级 / 归档 / 恢复
- 当前状态：[ ] / [/] / [?] / [x]（可选）
- 目标状态：[ ] / [/] / [?] / [x]（可选）
- 开始日期：YYYY-MM-DD（可选）
- 计划完成日期：YYYY-MM-DD（可选）
- 完成日期：YYYY-MM-DD（完成时可选，默认当天）
- 新项目：（可选）
- 新标题：（可选）
- 优先级：⏫ / 🔺 / 🔼 / 🔽（可选）
- 备注：
```

## 使用说明

- 如果任务标题不唯一，尽量带上项目名、父任务或日期
- 只要你说明新状态，我就按这个状态更新
- 如果状态变更伴随时间变化，可以一并提供日期
- 如果你只说“完成了”，我会默认改成 `[x]` 并补当天 `✅`
- 调整备注时，备注写在对应任务下方的普通列表行，不追加到 checkbox 任务行尾
- 完成任务不自动归档；必须明确说“归档”才移动到 `已归档.md`
