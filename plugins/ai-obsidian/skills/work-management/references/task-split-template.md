# 任务拆分模板

当一个父任务需要拆成多个子任务时，可以按下面格式发我：

```text
任务拆分：
- 父任务标题：
- 所属项目：
- 父任务开始日期：YYYY-MM-DD
- 父任务计划完成日期：YYYY-MM-DD
- 父任务状态：默认 [ ]
- 子任务列表：
	1. 子任务一
		- 开始日期：YYYY-MM-DD
		- 计划完成日期：YYYY-MM-DD
		- 状态：默认 [ ]
	2. 子任务二
		- 开始日期：YYYY-MM-DD
		- 计划完成日期：YYYY-MM-DD
		- 状态：默认 [ ]
```

## 使用说明

- 父任务只表达最终交付目标
- 子任务只表达具体可执行步骤
- 子任务尽量一条一个动作
- 如果子任务太少，我会提醒你继续补充
- 写入任务文件时，每一层任务层级使用 1 个 Tab 缩进，不使用空格缩进，并保留与父任务一致的 `#project/项目名`
