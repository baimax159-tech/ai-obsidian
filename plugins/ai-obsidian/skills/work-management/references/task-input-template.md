# 任务输入模板

你可以直接按下面格式发我：

```text
新增任务：
- 任务标题：
- 所属项目：
- 开始日期：YYYY-MM-DD
- 计划完成日期：YYYY-MM-DD
- 优先级：⏫ / 🔺 / 🔼 / 🔽（可选）
- 备注：
- 状态：默认未开始 [ ]
- 父任务：
- 子任务：
	- 
	- 
```

## 使用说明

- `所属项目`、`开始日期` 和 `计划完成日期` 必填
- 如果是大任务，先给父任务，再列子任务
- 子任务尽量写成可执行动作
- 如果暂时没有子任务，可以先只给父任务，但要说明后续是否要拆分
- 如果你只发一句话，我会先补齐缺失项再创建任务
- 写入任务文件时，每一层任务层级使用 1 个 Tab 缩进；备注放在对应任务下方的普通列表行，不追加到 checkbox 任务行尾
- 新任务写入 `<项目根目录>/任务/{YYYY}/计划中.md`
